> **布局要点**：右上角为两行结构 — 第 1 行是窗口控制按钮（○○○），第 2 行是「更多」胶囊按钮。
> 左侧标题区和中部快捷功能区的高度 = 窗口控制行 + 更多菜单行的总高度。

### 区域说明

#### 1. 左侧边栏 - 快捷方式（启动台）
- **功能定位**: 用户自定义的快速启动台，类似开始菜单
- **布局**: 单列垂直排列，固定宽度
- **溢出处理**: 纵向滚动
- **视觉风格**: 毛玻璃悬浮岛
- **详细设计**: 见下方「快捷方式栏」章节

#### 2. 顶部导航栏

##### 2.1 左侧 - 返回按钮 + 标题（共用悬浮岛）
- **主页状态**: 只显示标题（如"PGB1"或"主页"）
- **子页面状态**: `[← 返回] 页面标题`
- **返回逻辑**: 始终返回上一级
- **视觉**: 圆角悬浮岛，与标题在同一个容器内
- **高度**: 固定 `--floating-navbar-height`（100px），所有页面一致
- **标题字号**: `--text-4xl`

##### 2.2 中部 - 快捷功能区
- **功能定位**: 当前页面相关的应用内快捷操作
- **内容**: 随页面变化（详见"页面结构"章节）
- **布局**: 横向排列的按钮/图标
- **高度**: 比标题区略矮（顶部有 `--spacing-3` 出血），底边与标题区对齐
- **溢出处理**:
  - 设置最小窗口尺寸限制
  - 窗口过小时支持横向滚动（滚轮操作）
  - 优先保证核心功能可见
- **视觉**: 悬浮岛风格

##### 2.3 右侧 - 窗口控制 + 更多菜单（两行结构）

右上角为独立的两行区域：

- **第 1 行 — 窗口控制按钮**: 最小化、最大化、关闭
  - 位置：始终在窗口最右上角
  - 独立性：不属于任何悬浮岛，独立显示
  - 样式：圆形按钮，符合毛玻璃主题
- **第 2 行 — 更多菜单**: 悬浮岛风格的下拉按钮，位于窗口控制按钮正下方
  - 显示不常用操作按钮（如刷新、设置等，根据页面不同而变化）
  - 视觉：与其他悬浮岛统一圆角（`--floating-navbar-radius`）

**高度对齐**：左侧标题区和中部快捷功能区的总高度 = 窗口控制行 + 更多菜单行的总高度。

#### 3. 主功能区
- **功能**: 核心内容展示区域
- **内容**: 根据页面显示不同内容（卡片、列表、双视图等）
- **视觉**: 毛玻璃风格容器

```
    ↓ 点击项目卡片
```

### 快捷方式栏（左侧边栏）

用户自定义的快速启动台，支持添加应用、文件夹、网页三种快捷方式。

#### 布局

```
┌─────┐
│ [+] │  ← 添加按钮（始终在最顶部）
│─────│
│[图标]│  ← 快捷方式1
│ 名称 │
│─────│
│[图标]│  ← 快捷方式2
│ 名称 │
│─────│
│[图标]│  ← 快捷方式3
│ 名称 │
│ ... │
│     │  ← 超出时纵向滚动
└─────┘
```

#### 快捷方式类型

| 类型 | 图标来源 | 点击行为 |
|------|---------|---------|
| **应用** | 自动提取 exe 文件图标 | 启动应用程序 |
| **文件夹** | 系统文件夹图标 | 用系统文件浏览器打开 |
| **网页** | 自动获取网站 favicon | 用系统默认浏览器打开 |

#### 添加流程

点击 `[+]` 按钮后弹出添加弹窗：

```
┌──────────────────────────────────────┐
│ 添加快捷方式                    [×]  │
│ ─────────────────────────────────── │
│                                      │
│ 类型:                                │
│   ○ 应用    ○ 文件夹    ○ 网页      │
│                                      │
│ 路径/地址:                           │
│ [________________________] [浏览...] │
│                                      │
│ 名称:                                │
│ [________________________]           │
│                                      │
│ 图标预览: [🔲]                       │
│                                      │
│              [添加]  [取消]          │
└──────────────────────────────────────┘
```

**交互细节**：

1. **选择类型**：单选，默认不选（用户必须选一个）
2. **路径/地址**：
   - 应用：点击 `[浏览...]` 打开文件选择器，过滤 `.exe` 文件
   - 文件夹：点击 `[浏览...]` 打开文件夹选择器
   - 网页：手动输入 URL（隐藏 `[浏览...]` 按钮）
3. **名称**：
   - 应用：自动填充 exe 文件名（去掉 `.exe` 后缀），用户可修改
   - 文件夹：自动填充文件夹名，用户可修改
   - 网页：自动从网页 `<title>` 获取，获取失败则留空让用户填写
4. **图标预览**：
   - 应用：选择 exe 后自动提取并显示图标
   - 文件夹：显示系统默认文件夹图标
   - 网页：输入 URL 后自动获取 favicon 并显示（获取失败则显示默认地球图标）
5. **[添加] 按钮**：路径/地址和名称都填写后才可点击，否则置灰

#### 图标获取策略

##### 应用图标
- **方式**：Rust 后端调用 Windows API 提取 exe 文件的图标资源
- **缓存**：提取后缓存为 PNG 保存到应用数据目录，避免重复提取
- **失败兜底**：显示默认应用图标（齿轮图标或类似）

##### 文件夹图标
- **方式**：使用统一的文件夹图标（从系统图标库提取或内置）
- 不需要为每个文件夹单独获取图标

##### 网页 Favicon
- **方式**：
  1. 优先尝试 `{domain}/favicon.ico`
  2. 失败则请求网页 HTML，解析 `<link rel="icon">` 标签获取 favicon 地址
  3. 下载 favicon 并缓存为 PNG
- **缓存**：保存到应用数据目录，以域名为文件名
- **失败兜底**：显示默认地球图标

#### 快捷方式管理

##### 右键菜单
右键点击已有快捷方式，弹出菜单：
- **编辑**：打开编辑弹窗（同添加弹窗，预填当前数据）
- **删除**：确认后删除（无需二次确认弹窗，直接删除即可）

##### 排序
- **拖拽排序**：用户可拖拽快捷方式图标调整顺序
- 拖拽时有视觉反馈（被拖拽项半透明，插入位置有指示线）

#### 溢出处理
- 快捷方式数量超出可见区域时，支持纵向滚动
- 滚动条样式符合毛玻璃主题（半透明细滚动条）
- `[+]` 按钮始终在最顶部（不随滚动移动）

#### 主功能区结构

##### 小标题栏（主功能区顶部）

**正常模式**：
```
┌─────────────────────────────────────────────────────┐
│  素材列表  [🌲树形视图] [📄名称视图] [🔄刷新] [多选]  │
└─────────────────────────────────────────────────────┘
```

**多选模式**：
```
┌──────────────────────────────────────────────────────────────┐
│  素材列表  [🌲树形视图] [📄名称视图] [🔄刷新] [多选✓] [全选]  │
└──────────────────────────────────────────────────────────────┘
```

**点击全选后，按钮文字切换**：
```
┌──────────────────────────────────────────────────────────────────┐
│  素材列表  [🌲树形视图] [📄名称视图] [🔄刷新] [多选✓] [取消全选]  │
└──────────────────────────────────────────────────────────────────┘
```

**按钮说明**：

**所有模式下始终显示**：
- **小标题**: "素材列表" 或当前视图名称
- **[🌲树形视图]** - 切换到树形分类视图（当前激活的视图高亮显示）
- **[📄名称视图]** - 切换到名称排序视图（当前激活的视图高亮显示）
- **[🔄刷新]** - 重新扫描文件系统，更新列表

**正常模式下**：
- **[多选]** - 进入多选模式（用于批量上传）

**多选模式下**：
- **[多选✓]** - 点亮状态，再次点击退出多选模式并清除所有选择
- **[全选]** - 选中当前视图中的所有素材，点击后按钮文字变为 `[取消全选]`
- **[取消全选]** - 取消选中所有素材，点击后按钮文字变回 `[全选]`

##### 视图内容区

**模式1：树形分类视图**
```
功能分类1（如Ambient）
  ├─ 静帧
  │   ├─ [100%] - 3个文件
  │   ├─ [70%] - 2个文件
  │   └─ [50%] - 1个文件
  └─ 序列帧
      ├─ [100%] - 5组
      └─ [70%] - 3组
```
- **层级展示**: 文件类型 → 缩放比例 → 具体文件
- **展开/折叠**: 支持节点展开/折叠
- **用途**: 整理、分类、批量操作

**模式2：名称排序视图**
```
┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐
│ [卡片1]          │ │ [卡片2]          │ │ [卡片3]          │
│ 素材名称A        │ │ 素材名称B        │ │ 素材名称C        │
│ [静帧] [75%]     │ │ [序列帧] [100%]  │ │ [静帧] [50%]     │
│ 缩略图...        │ │ 缩略图...        │ │ 缩略图...        │
└──────────────────┘ └──────────────────┘ └──────────────────┘
```
- **扁平展示**: 所有文件按名称排序
- **卡片样式**: 与树形视图使用同一种素材卡片（→ 参考 `卡片设计.md` - 素材卡片）
- **无需额外字段**: 任务页本身就是单个任务的页面，所有素材都属于同一任务，不需要显示分类信息
- **排序规则**: 按文件名字母/拼音顺序
- **用途**: 快速查找、预览、拖拽上传

#### 功能行为说明

**[子任务 X/Y]**（仅有子任务时显示）:
- 点击弹出子任务管理弹窗
- **弹窗内容**：
  - 显示所有子任务列表
  - 每个子任务有复选框（已完成/未完成）
  - 用户可手动勾选/取消勾选
  - 底部显示"确定"和"取消"按钮
- **自动判定**：
  - 页面刷新时，如果检测到所有素材都上传
  - 自动弹出此弹窗，询问"哪个子任务完成了？"
  - 用户选择后更新进度
- **数据同步**：修改后立即更新任务卡片的"状态 X/Y"显示

**[规范化]**: 一键整理 `00_original/` 目录中的AE导出文件
  - **识别逻辑**：
    - 只有 `xxx_01.ext` → 静帧（去掉 `_01` 后缀）
    - 有 `xxx_01.ext`, `xxx_02.ext` ... → 序列帧（创建文件夹并移动）
  - **详细规则** → 参考 `文件命名与组织规则.md` - 步骤2：规范化操作

**[缩放]**: 批量缩放静帧（序列帧不参与）
  - **操作流程**：
    - 用户选择规范化后的静帧
    - 指定缩放比例（预设：100%/70%/50%/40%，支持自定义）
    - 程序执行缩放，生成到 `01_scale/[XX]/`
  - **算法**：高质量缩放（Lanczos或Bicubic）
  - **详细规则** → 参考 `文件命名与组织规则.md` - 步骤3：缩放处理

**[转换]**: 批量格式转换（静帧 + 序列帧）
  - **用途**：将 `01_scale/` 中的静帧和序列帧转换为webp格式（静帧用imagine，序列帧用TexturePacker）
  - **触发流程**：点击 `[转换]` 按钮 → 弹出素材选择窗口
  - **详细流程** → 参考 `文件命名与组织规则.md` - 步骤3.5：格式转换

- **序列帧完成判定**：`序列帧已整理数 == 序列帧总数`
- **全部完成判定**：
  - 如果只有静帧：静帧全部完成
  - 如果只有序列帧：序列帧全部完成
  - 如果两者都有：静帧 AND 序列帧全部完成
- **自动清理**：
  - 等待2秒（确保所有文件都已处理）
  - 关闭所有相关进程和窗口
  - 关闭悬浮窗
  - 显示完成提示："转换完成！静帧 X 个，序列帧 Y 个"

**手动完成**：

- 用户可以随时点击 `[完成转换]` 提前结束
- 执行相同的清理操作

**清理操作**：

- 停止文件监控
- 关闭imagine进程（如果启动过）
- 关闭TexturePackerGUI进程（如果有未关闭的）

- 关闭imagine打开的文件浏览器（通过窗口标题匹配）

### 打卡自动化流程（Tauri WebView 方案）

#### 技术方案

使用 Tauri 的 `WebviewWindow` 创建后台不可见的浏览器窗口，通过 `evaluate_script()` 注入 JavaScript 完成页面交互。

**优势**（相比旧系统的 Selenium）：
- 无需安装 Chrome / ChromeDriver
- 无版本兼容问题
- 轻量，启动快
- 与 PGB1 应用一体化

#### 出勤/退勤打卡流程

```
1. 创建隐藏的 WebviewWindow（不显示）
2. 导航到打卡网站 URL
3. 等待页面加载完成
4. JS注入：查找账号/密码输入框 → 自动填写
5. JS注入：查找「ログイン」按钮 → 点击
6. 等待页面跳转（URL 变化）
7. JS注入：查找「打刻」链接 → 点击
8. 等待页面跳转
9. JS注入：查找「出勤」或「退勤」按钮
   - 如果按钮已禁用（disabled）→ 返回"已打卡"
   - 如果按钮可用 → 点击
10. 等待操作完成（2秒）
11. 返回结果（成功/失败）→ 弹出结果确认弹窗
```

**错误处理**：
- 每步设置超时（默认30秒）
- 元素找不到 → 返回失败，提示用户手动打卡
- 网络错误 → 返回失败，提示检查网络
- 按钮已禁用 → 视为已打卡，返回成功

**WebView 生命周期**：
- 打卡开始时创建
- 用户点击「确认完成」时销毁
- 用户点击「查看结果」时前台显示，用户关闭后销毁

#### 日报打开流程

```
1. 创建可见的 WebviewWindow（前台显示）
2. 导航到日报网站 URL
3. 等待页面加载完成
4. JS注入：window.scrollTo(0, document.body.scrollHeight)
   （模拟 Ctrl+End，跳转到页面底部）
5. WebView 保持打开，用户手动填写
6. 用户关闭 WebView 窗口即可
```

**注**：日报 WebView 不自动关闭，不弹确认窗口。

## 🌐 翻译功能

### 功能概述

全局快捷键呼出的独立置顶翻译窗口，用于工作中快速翻译中英文沟通消息。调用 Gemini API 实现翻译，支持对话模式（双向自动检测）和输出模式（固定目标语言）。

**使用场景**：用户在 AE、浏览器等任意软件中工作时，需要翻译与新加坡/马来西亚同事的英文工作消息。

**参考旧系统**：`D:\work\pgsoft\translator`（Electron + Gemini API）

---

### 呼出方式

**全局快捷键**：按下快捷键 → 翻译窗口显示/隐藏切换（toggle）

**快捷键设置**：
- 默认快捷键：`Ctrl+Shift+T`（开发阶段可调）
- 在应用设置中可自定义快捷键
- 提供一个特殊选项：「使用计算器键」（拦截键盘上的 Calculator 多媒体键）

**技术实现**：
- Rust 后端通过 Windows API `RegisterHotKey` 注册全局热键
- 计算器键对应 `VK_LAUNCH_APP2`，注册后系统级拦截（不再打开计算器）
- PGB1 未运行时，计算器键恢复原功能

**行为**：
- 快捷键按下 → 窗口不存在则创建并显示，已存在则 toggle（显示↔隐藏）
- 不是每次新建窗口，是同一个窗口的 show/hide
- 窗口隐藏时不销毁，保留输入/输出内容

### 功能详细设计

#### 翻译模式

**只有一种模式**：双语自动检测。

- 用户设定一组语言对（如 中文 ↔ English）
- 输入中文 → 自动翻译为英文
- 输入英文 → 自动翻译为中文
- API 自动检测输入语言，翻译为另一种
- 不需要手动切换方向，不需要选模式

#### 翻译风格

**固定风格**：简洁、阳光、亲切。不提供风格选项，Prompt 中固定：

> 翻译语气要求：简洁自然，阳光亲切，像同事之间日常沟通的感觉。不要罗嗦，不要过于正式。

#### 语言选项

底部语言对选择器，两个下拉框：

| 语言 | 代码 |
|------|------|
| 中文 | zh-CN |
| English | en |
| 日本語 | ja |

- 默认语言对：中文 ↔ English
- 用户可切换（比如改为 中文 ↔ 日本語）
- 语言列表精简为工作中实际用到的三种，未来可扩展

#### 输入/输出交互

- **[📋] 粘贴**：读取剪贴板内容填入输入框
- **[📄] 复制**：将输出框内容复制到剪贴板，按钮短暂变为 ✓ 反馈
- **[▶ 翻译]**：发送翻译请求，输出框显示"翻译中..."，完成后显示结果
- **Enter 快捷键**：在输入框中按 `Ctrl+Enter` 触发翻译（等同点击翻译按钮）
- 翻译过程中按钮置灰，防止重复请求

### 翻译 API 调用

**API 端点**：
```
POST https://generativelanguage.googleapis.com/v1beta/models/{modelName}:generateContent
```

**请求头**：
```
Content-Type: application/json
X-goog-api-key: {apiKey}
```

**Prompt 构造**：

```
You are a translator between {langA} and {langB}.
Detect the input language and translate to the other one.
Tone: concise, friendly, natural — like casual coworker chat. No fluff.
Only output the translation, nothing else.

"{text}"
```

**调用位置**：Rust 后端发起 HTTP 请求（通过 `reqwest` crate），避免前端直接暴露 API Key。

**错误处理**：
- API Key 未设置 → 输出框显示"请先在设置中配置 API Key"

## 🎨 视觉设计规范

### 毛玻璃风格（Glassmorphism）
- **背景**: 半透明模糊效果（`backdrop-filter: blur()`）
- **所有配置必须在 `DesignSystem.css` 中定义**
- **禁止硬编码颜色、透明度、模糊度**

### 悬浮岛（Floating Islands）
- 顶部导航栏的各区域均为独立的圆角悬浮岛
- 左侧边栏为悬浮岛

### 布局结构

**完整布局**（适用于图片、视频、PSD、序列帧等）：
```
┌─────────────────────────┐
│ 详情                    │ ← 标题
├─────────────────────────┤
│                         │
│                         │
│     预览区域            │ ← 根据文件类型显示不同内容
│                         │    （图片查看器、视频播放器、缩略图等）
│                         │
├─────────────────────────┤
│ 基本信息                │
│ ━━━━━━━━━━━━━━━━━━━━━  │
│ 文件名: xxxx.png        │
│ 缩放比例: 100%          │
│ 帧率: 30fps             │
│ 大小: 1.2MB             │
├─────────────────────────┤
│ 其他版本                │
│ ━━━━━━━━━━━━━━━━━━━━━  │
│ [格式] 100% - 原始      │ ← 点击格式按钮打开文件
│ [格式]  70% - 已缩放    │    Hover有高亮
│ [格式]  50% - 已完成    │
│ [格式]  已上传          │
└─────────────────────────┘
```

**简化布局**（适用于文档类：.txt / .doc / .pdf）：
```
┌─────────────────────────┐

├─────────────────────────┤
│                         │
│                         │

│                         │
│                         │
│                         │
```

### 内容区域

#### 1. 标题区
- 固定标题："详情"
- 样式：符合毛玻璃主题

#### 2. 预览区域

**根据文件类型提供不同的预览方式**：

##### 2.1 文档类文件（.txt / .doc / .pdf）

**预览方式**：
- 在预览区域直接渲染文档内容
- 每页纵向排列
- 宽度自适应侧边栏宽度
- 支持垂直滚动查看全部内容
- 支持一定程度的放大（缩放控制）

**布局简化**：
- ✅ **预览区域**：显示文档内容
- ❌ **不显示**：基本信息区、其他版本区（文档类文件不需要这些）

**交互**：
- 滚轮滚动查看文档
- 放大/缩小按钮或快捷键（待细调）

##### 2.2 视频文件（.mp4 / .mov / .avi 等）

**预览方式**：
- 在预览区域直接内嵌视频播放器
- 宽度自适应侧边栏宽度
- 高度根据视频比例自适应

**播放控制**：
- 播放按钮：点击开始/暂停播放
- 进度条：显示播放进度，可拖拽跳转
- 快捷键支持：
  - `←/→`：快进/后退（时间待定，建议 5秒或 10秒）
  - `Ctrl + ←/→`：单帧前进/后退

**布局**：
- ✅ **预览区域**：视频播放器
- ✅ **基本信息区**：文件名、帧率、分辨率、大小、时长
- ✅ **其他版本区**：显示（如果有多个工作流版本）

##### 2.3 图片文件（.png / .jpg / .jpeg 等）

**预览方式**：
- 加载原始图片（非缩略图）
- 初始状态：适应预览区域宽度/高度（保持比例）

**交互功能**：
- **滚轮缩放**：在预览区域内滚轮放大/缩小
- **拖拽移动**：放大后可拖拽查看图片其他部分
- **重置按钮**：恢复到初始适应大小（可选）

**布局**：
- ✅ **预览区域**：图片查看器
- ✅ **基本信息区**：文件名、分辨率、大小
- ✅ **其他版本区**：显示（如果有多个工作流版本）

##### 2.4 PSD文件（.psd）

**预览方式**：
- 显示PSD文件的缩略图（放大版）
- 宽度自适应侧边栏宽度

**特殊按钮**：
- **[打开PSD]** 按钮：位于预览区域内或下方
- 点击后在系统默认程序（通常是Photoshop）中打开

**布局**：
- ✅ **预览区域**：缩略图 + 打开按钮
- ✅ **基本信息区**：文件名、分辨率、大小
- ✅ **其他版本区**：显示（如果有多个工作流版本）

##### 2.5 其他文件类型（序列帧、精灵图等）

**预览方式**：
- 显示缩略图或动画预览
- 详细规则 → 参考 `卡片设计.md` - 预览图生成规则

**布局**：
- ✅ **预览区域**：缩略图/动画
- ✅ **基本信息区**：文件名、缩放比例、帧率、大小
- ✅ **其他版本区**：显示

#### 3. 基本信息区
- **文件名**：完整文件名（含扩展名）
- **缩放比例**：如 100%、70%、50%
- **帧率**：仅序列帧/视频显示（如 30fps）
- **大小**：文件大小，自动单位转换（KB/MB/GB）

#### 4. 其他版本区

**显示规则**：
- ✅ **显示**：图片、视频、PSD、序列帧等素材文件
- ❌ **不显示**：文档类文件（.txt / .doc / .pdf）

**显示内容**：该素材在不同工作流阶段的所有版本
- **包含**：
  - 所有工作流阶段（00_original、01_scale、02_done、nextcloud）
  - 所有缩放比例（100%、70%、50%等）
- **排序**：按工作流顺序排列
  ```
  00_original/100% - 原始
  01_scale/100% - 已缩放
  01_scale/70% - 已缩放
  01_scale/50% - 已缩放
  02_done/100% - 已完成
  02_done/70% - 已完成
  nextcloud/100% - 已上传
  ```
- **列表项格式**：`[格式图标] 缩放比例 - 状态文字`
- **交互**：
  - 点击格式图标按钮 → 打开该文件（注意：不是点击整行，只有格式按钮可点击）
  - Hover格式按钮 → 高亮显示

#### 5. 文件类型与布局对照表

| 文件类型 | 预览方式 | 基本信息区 | 其他版本区 | 特殊功能 |
|---------|---------|-----------|-----------|---------|
| **文档类**（.txt/.doc/.pdf） | 直接渲染文档内容，纵向排列，可滚动 | ❌ 不显示 | ❌ 不显示 | 支持缩放 |
| **视频**（.mp4/.mov/.avi） | 内嵌播放器 | ✅ 显示 | ✅ 显示 | 播放控制、快捷键 |
| **图片**（.png/.jpg/.jpeg） | 加载原始图片 | ✅ 显示 | ✅ 显示 | 滚轮缩放、拖拽移动 |
| **PSD**（.psd） | 缩略图放大 | ✅ 显示 | ✅ 显示 | 打开PSD按钮 |
| **其他**（序列帧/精灵图等） | 缩略图/动画 | ✅ 显示 | ✅ 显示 | - |

风格关键词： 工业科幻控制面板UI，深色主题，高对比冷蓝强调色

视觉配比（6:3:1）：
60% 深灰黑 — 面板底色、背景、大面积填充。中性灰黑，不带蓝色偏移
30% 高对比蓝 — 边框亮线、激活态、强调元素、LED发光点缀。面积小但亮度高
10% 白色 — 主文本、关键数据、图标高光。在深黑底上形成硬朗的阅读对比

配色体系：

背景底色：#0C0D10 ~ #18191E（纯净深灰黑，RGB 三通道接近等值，无蓝色偏移）
主强调色：#4A9EF5 ~ #6BB8FF（中等饱和度的科技蓝，仪表盘指示灯的冷白蓝）
高亮/激活态：#8ECFFF（亮冰蓝，用于选中边框和关键数据）
次级强调：#2D5A8A（暗蓝，用于非活跃但仍需标识的元素，如未选中的标签页底色）
文本主色：#E2E4E8（近白灰，在深黑背景上保证高对比度可读性）
文本次级：#8A8D96（中灰，明确低于主色但不会消失）
非活跃元素：#2E3036（深灰，和背景只有微妙的明度差）
危险/警告色：#E05A5A（偏暗的冷红，不是橙红）
成功/确认色：#4ECBA0（低饱和青绿）
品质色阶：维持绿→蓝→紫→金，但整体饱和度压低，金色改为 #C9A84C 避免和蓝色体系冲突

毛玻璃面板：

填充色：rgba(14, 15, 18, 0.72) ~ rgba(22, 23, 28, 0.60)（中性深灰黑底，无蓝色色偏）
背景模糊：backdrop-filter: blur(18px~24px)
边框：1px，rgba(100, 180, 255, 0.10) ~ rgba(100, 180, 255, 0.18)（冷蓝亮线，这是蓝色占比的主要载体之一——精准、锐利、不扩散）
圆角：4px~8px 或直角，硬边矩形
噪点纹理：叠加极低不透明度的冷色噪点（2%~5%），增强工业磨砂质感
面板底色严格保持中性灰黑，蓝色只出现在边框线和发光点上，不做大面积蓝色wash

底层毛玻璃（应用根背景层）：

之前状态：半透明接近无色，几乎不可感知
优化方向：给底层添加一层轻微的深灰调（rgba(10, 11, 14, 0.35)），配合 blur(12px) 和 2% 噪点叠加，使底层从"完全透明"变为"有质感的深色雾面"，让上层面板与背景之间有更明确的层次区分

层级规则（高对比度明暗阶梯）：

最底层（背景）：纯深黑 #0C0D10 + 3D场景内容
根层毛玻璃：rgba(10, 11, 14, 0.35)，blur 12px — 给背景增加质感锚点
第一层面板：rgba(22, 23, 28, 0.60)，blur 24px，边框 rgba(100, 180, 255, 0.10) — 主功能区
第二层面板（浮于第一层之上）：rgba(14, 15, 18, 0.72)，blur 18px，边框 rgba(100, 180, 255, 0.14) — 更密实的深色面板
弹窗/模态框：rgba(12, 13, 16, 0.82)，blur 16px，边框 rgba(100, 180, 255, 0.18)，box-shadow: 0 0 24px rgba(74, 158, 245, 0.10) — 最明显的蓝色发光边缘
层级靠面板明度跳跃 + 边框亮度梯度拉开。底层暗、上层相对更暗更密实，形成纵深感。不用 Material 式的大阴影

交互态：

默认态：中性深灰黑面板 + 暗蓝边框
悬停态：边框亮度提升至 rgba(100, 180, 255, 0.28)，面板内部微微提亮 2%
选中/激活态：边框变为 #6BB8FF（实色或高不透明度），面板底色叠加一层 rgba(74, 158, 245, 0.08) 的蓝色wash，关键文字变为强调蓝或纯白
禁用态：面板和文字整体压暗至 35% 不透明度

排版： 等距/几何无衬线西文字体（DIN、Eurostile、Bender 方向），中文黑体。信息密度高，颜色层级优先于留白。中英混排保留。数字和数据用等宽字体。

整体气质： 高对比工业控制终端。黑底是绝对的主角（60%），蓝色只作为精准的"激光线"和"LED指示灯"存在（30%），白色是稀缺的高亮点缀（10%）。深灰黑面板上，冷蓝色边框线像是从面板缝隙中透出的蓝色LED背光。面板表面叠加微弱噪点，像是磨砂金属面板在低照度下的质感。安静、精密、硬朗、有力量感。
